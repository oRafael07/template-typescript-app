# App

Template Base for Typescript Applications

# Stacks

- Typescript
- Tsx
- Eslint
- Tsup
- Prettier

# Running

## Clone repository
```
$ git clone https://github.com/oRafael07/template-typescript-app.git
```

## Install Dependencies
```
$ npm install
```

## Scripts

### run develop mode
```
$ npm run dev
```
### build application
```
$ npm run build
```
### run production mode
```
$ npm run start
```
### run lint fixes
```
$ npm run lint
```
