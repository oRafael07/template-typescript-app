# App

Template Base for Typescript Applications

# Stacks

- Typescript
- Tsx
- Eslint
- Tsup
- Prettier

# Running

## Clone repository
```bash
git clone https://github.com/oRafael07/template-typescript-app.git
```

## Install Dependencies
```bash
npm install
```

## Scripts

### run develop mode
```bash
npm run dev
```
### build application
```bash
npm run build
```
### run production mode
```bash
npm run start
```
### run lint fixes
```bash
npm run lint
```
